API Endpoints

GET /api/v1/sources

List all available sources

GET /api/v1/sources/{id}

Get source details

GET /api/v1/sources/{id}/search

Search for books in a source

GET /api/v1/sources/{id}/popular

Get popular books from a source

GET /api/v1/books

Get your library

GET /api/v1/books/{id}/chapters

Get chapters for a book

GET /api/v1/chapters/{id}/pages

Get pages for a chapter

GET /health

Health check endpoint
