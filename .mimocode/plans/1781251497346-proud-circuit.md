# Leaderboard: User Books + Full Profile Screen

## Goal
Show synced books on leaderboard user profiles, allow opening them if source is installed, and create a beautiful full-screen user profile view.

---

## 1. Add Synced Books to LeaderboardEntry

**File:** `domain/.../models/entities/LeaderboardEntry.kt`

Add a `syncedBooks` field:
```kotlin
data class LeaderboardEntry(
    ...existing fields...,
    val syncedBooks: List<SyncedBookSummary> = emptyList()
)

data class SyncedBookSummary(
    val title: String,
    val coverUrl: String,
    val sourceName: String,
    val sourceId: Long,
    val bookUrl: String,
)
```

## 2. Fetch User's Synced Books from Supabase

**File:** `data/.../leaderboard/LeaderboardRepositoryImpl.kt`

Add a method to fetch a user's synced books:
```kotlin
suspend fun getUserSyncedBooks(userId: String): List<SyncedBookSummary>
```
Query `synced_books` table filtered by `user_id`, return top 6 books sorted by `last_read DESC`.

**File:** `domain/.../usecases/leaderboard/LeaderboardUseCases.kt`

Add `getUserSyncedBooks(userId)` use case that delegates to the repository.

## 3. ViewModel: Load Books When User Clicks a Ranker

**File:** `presentation/.../leaderboard/LeaderboardViewModel.kt`

Add:
- `loadUserBooks(userId: String)` — fetches synced books for a user
- State field: `selectedUserBooks: List<SyncedBookSummary>`

## 4. New Full-Screen User Profile Screen

**File:** `presentation/.../leaderboard/UserProfileScreen.kt` (NEW)

Replace the bottom sheet with a full-screen profile view. Layout:

### Header Section
- Gradient background with user avatar (large, with level ring)
- Username, level title, tier badge
- Rank badge (#N), reading time

### Stats Row
- Chapters read, Books completed, Streak, Reading time
- Each in a card with icon + value + label

### XP Progress
- Level progress bar with XP values

### Synced Books Section (NEW)
- Grid of book covers (2 columns, max 6 shown)
- Each book shows: cover image, title, source name
- If user taps a book:
  - Check if source is installed via `catalogStore.get(sourceId)`
  - If installed → open book (like PopularBooks does)
  - If not installed → show "Source not installed" message

### Close button to go back to leaderboard

## 5. Book Opening Logic

**Reuse pattern from `PopularBooksViewModel.openSource()`:**
1. Check `bookRepository.findDuplicateBook(title, sourceId)` for local
2. If local exists → navigate to bookDetail
3. If source installed → create Book, upsert, navigate to bookDetail
4. If source not installed → show snackbar/dialog

## 6. Navigation

**File:** `presentation/.../core/NavigationRoutes.kt`

Add: `const val userProfile = "userProfile/{userId}"`

**File:** `presentation/.../core/CommonNavHost.kt`

Register the new route with `UserProfileScreen`.

**File:** `presentation/.../leaderboard/LeaderboardScreen.kt`

Change: clicking a rank row navigates to `userProfile/{userId}` instead of showing bottom sheet.
