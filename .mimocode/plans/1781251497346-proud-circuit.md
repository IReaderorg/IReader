# Gamification Integration Improvements

## Goal
Improve user engagement by integrating gamification deeper into the reading experience: in-reader feedback, reading challenges, milestone celebrations, achievement progress, and engagement hooks.

## 1. In-Reader Gamification Overlay

Show real-time gamification feedback while reading.

**Files to modify:**
- `presentation/.../reader/components/ReadingTimeIndicator.kt` — extend to show XP gains
- `presentation/.../reader/viewmodel/ReaderViewModel.kt` or `ReaderStatisticsViewModel.kt` — add XP observation
- `domain/.../usecases/statistics/TrackReadingProgressUseCase.kt` — calculate XP earned

**Implementation:**
- Add a mini overlay (bottom-right, semi-transparent) showing:
  - Session XP earned (updates every 30s when reading time saves)
  - Current streak status (fire icon + days)
  - Level progress mini-bar
- Show a brief "Level Up!" animation when level increases (detected via `getStatisticsFlow()`)
- Show "+N XP" floating text on chapter completion (80% threshold)

**New files:**
- `presentation/.../reader/components/ReaderGamificationOverlay.kt` — composable overlay

## 2. Reading Challenges (Weekly/Monthly Goals)

Add configurable reading goals with spirit stone rewards.

**Files to create:**
- `domain/.../models/gamification/ReadingChallenge.kt` — data models (WeeklyGoal, MonthlyGoal)
- `domain/.../data/repository/ReadingChallengeRepository.kt` — interface
- `data/.../challenge/ReadingChallengeRepositoryImpl.kt` — local persistence (SQLDelight or preferences)
- `presentation/.../readinghub/ReadingChallengeCard.kt` — UI card
- `presentation/.../readinghub/ReadingChallengeViewModel.kt` — state management

**Implementation:**
- User sets daily/weekly reading time goal (e.g., 30 min/day, 3 hrs/week)
- Progress tracked via existing `getStatisticsFlow()`
- On goal completion: reward spirit stones (10-50 based on goal difficulty)
- Show progress ring on Reading Hub screen
- Persist goals in local preferences

**Modify:**
- `presentation/.../readinghub/ReadingHubScreen.kt` — add challenge card section

## 3. Milestone Celebrations

Special celebrations for significant achievements.

**Files to create:**
- `presentation/.../core/ui/MilestoneCelebrationOverlay.kt` — full-screen celebration animation

**Files to modify:**
- `presentation/.../readinghub/ReadingHubViewModel.kt` — detect milestones
- `presentation/.../readinghub/ReadingHubScreen.kt` — show celebration overlay

**Milestones to celebrate:**
| Milestone | Reward | Effect |
|-----------|--------|--------|
| 10 books read | 50 💎 | Confetti + badge |
| 50 books read | 200 💎 | Gold confetti + title |
| 100 books read | 500 💎 | Diamond confetti + rare title |
| 1000 chapters | 100 💎 | Star burst |
| 10,000 chapters | 500 💎 | Legendary celebration |
| 7-day streak | 20 💎 | Fire animation |
| 30-day streak | 100 💎 | Flame wall |
| 365-day streak | 1000 💎 | Legendary fire |

**Implementation:**

- On statistics update, check if any milestone threshold crossed
- Show full-screen overlay with animation (Lottie or Compose animation)
- Award spirit stones via `gamificationRepository.spendStones()` or local credit
