# JavaScript Plugin System

IReader supports a powerful plugin system that allows developers to create sources using JavaScript. This enables the community to add support for new novel sites without updating the app itself.

## Getting Started
To start developing plugins, check out the [Plugin Development Guide](plugin-development/getting-started.md).

## Key Concepts
- **Manifest**: Defines plugin metadata (id, name, version). See [Manifest Format](plugin-development/manifest-format.md).
- **Plugin Types**: Understand the different types of plugins available. See [Plugin Types](plugin-development/plugin-types.md).
- **API Reference**: Detailed documentation of the available APIs. See [API Reference](plugin-development/api-reference.md).

## Testing
Learn how to test your plugins locally before publishing. See [Testing Guide](plugin-development/testing.md).

## Publishing
Ready to share your plugin? Follow the [Submission Guidelines](plugin-development/submission-guidelines.md).
