# Features Guide

Explore the advanced features of IReader, from AI translation to community engagement.

## AI Translation

IReader supports powerful AI translation engines to translate novels into your language.

### Supported Engines
*   **Google Gemini**: High-quality, fast translation (Requires API Key).
*   **OpenAI (ChatGPT)**: Industry-standard AI translation (Requires API Key).
*   **DeepSeek**: Advanced coding and text model (Requires API Key).
*   **Gradio/HuggingFace**: Connect to custom spaces.

### Setting up Gemini Translation
1.  Go to **Settings** → **General** → **Translator**.
2.  Tap **Translation Engine** and select **Google Gemini**.
3.  **API Key**: You need a Google Gemini API key.
    *   Get one for free at [Google AI Studio](https://aistudio.google.com).
4.  Enter your API Key in the field.
5.  **Model**: Select a model (e.g., `gemini-pro`).

## Reviews

Share your thoughts on novels with the community.

### Writing a Review
1.  Open a book's detail page.
2.  Scroll down to the **Reviews** section.
3.  Tap the **Write Review** button (pencil icon).
4.  Rate the book (1-5 stars) and write your review.
5.  Tap **Submit**.

> [!NOTE]
> You may need to be logged in or have a configured backend to submit reviews.

## Badges

Earn badges for your activity in the app.

### How to Earn Badges
*   **Reviewer**: Write reviews for books.
*   **Supporter**: Donate to the project.
*   **NFT Owner**: Verify ownership of an IReader NFT.
*   **Translator**: Contribute translations.

To view your badges, go to **Settings** → **Badges**.

## Custom Fonts

Customize your reading experience with your favorite fonts.

1.  Go to **Settings** → **Appearance**.
2.  Tap **Font**.
3.  Select a built-in font or tap **Add Font** to use a custom `.ttf` or `.otf` file from your device.
