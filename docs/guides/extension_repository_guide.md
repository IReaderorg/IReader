## What is an Extension Repository?

An extension repository is a collection of sources (plugins) that let you browse and read novels from different websites. Think of it like an app store, but for novel sources!

## Quick Setup (Recommended)

### Step 1: Enable JavaScript Plugins

1. Open **IReader**
2. Go to **Settings** â†’ **General**
3. Find **"Enable Javascript Plugin"** and turn it **ON**

![Enable JS Toggle](../../screenshots/enable-js.png)

### Step 2: Add the Official IReader Repository

1. Go to **Settings** â†’ **Repository**
2. Tap the **+ (Add)** button
3. Select **"Add Custom Repository"**

   ```
   https://raw.githubusercontent.com/IReaderorg/IReader-extensions/master/index.min.json
   ```

## Troubleshooting

### "No sources found" after adding repository

1. Make sure **JavaScript Plugins** is enabled in Settings â†’ General
2. Check your internet connection
3. Try refreshing the sources list
4. Restart the app and try again

### Repository won't add

1. Double-check the URL is correct
2. Make sure you have internet access
3. The repository server might be temporarily down - try again later

### Extensions not showing up

1. Check the language filter - you might have filtered them out
2. Go to Sources and tap the filter icon
3. Select **All Languages** or your preferred language

### "Untrusted extension" warning

This is normal for new extensions. IReader warns you before running code from external sources.

1. Only install from trusted repositories (like LNReader)
2. You can manage trusted extensions in **Settings** â†’ **Extensions** â†’ **Trust Management**

## Storage Options

By default, extensions are saved to the app's cache (no permissions needed).

### To save extensions to external storage:

1. Go to **Settings** â†’ **General**
2. Disable **"Saved Sources to Cache"**
3. Grant storage permissions when prompted

This makes it easier to backup and manage your extensions manually.

## FAQ

**Q: Do I need to add a repository to use IReader?**
A: Yes, IReader no longer includes a default repository. You need to add at least one repository to access novel sources.

**Q: Which repository should I use?**
A: We recommend adding **both**:
- **IReader Extensions** (Official) - Native extensions with better performance
- **LNReader** - 100+ JavaScript sources for maximum coverage

**Q: Can I add multiple repositories?**
A: Yes! You can add as many repositories as you want.

**Q: Are the extensions safe?**
A: Extensions from official repositories (like LNReader) are community-maintained and safe. Only add repositories from sources you trust.

**Q: How do I update my extensions?**
A: Go to Settings â†’ Extensions â†’ JavaScript Plugins and tap "Check for Updates", or enable auto-updates.
