### Paggamit ng LNReader Sources (Inirerekomenda)

Sinusuportahan na ngayon ng IReader ang mga LNReader source! Sundin ang mga hakbang na ito:

1. **I-enable ang JavaScript Plugins**
   - Pumunta sa **Settings â†’ General**
   - I-toggle ang **"Enable Javascript Plugin"**

2. **Magdagdag ng LNReader Repository**
   - Pumunta sa **Settings â†’ Repository**
   - I-tap ang **Add icon (+)**
   - Piliin ang **Quick Add Popular Repo**
   - Piliin ang **LNReader**
   - I-save at i-refresh ang remote sources sa Source screen

3. **Mga Pahintulot sa Storage (Opsyonal)**
   - Bilang default, ang mga LNReader source ay naka-save sa app cache (walang kinakailangang pahintulot)
   - Para i-save ang mga source sa external storage para sa mas madaling access, i-disable ang **"Saved Sources to Cache"** sa Settings â†’ General
   - Kung gumagamit ng external storage, ibigay ang mga pahintulot sa storage kapag hiniling

4. **Magsimulang Magbasa**
   - Mag-browse ng mga source para makahanap ng mga novel
   - Magdagdag ng mga libro sa iyong library
   - Mag-download ng mga chapter para sa offline na pagbabasa

> **Tandaan:** Hindi na sinusuportahan ng IReader ang in-app default repository. Kailangan mong manu-manong magdagdag ng mga repository sa Settings.
